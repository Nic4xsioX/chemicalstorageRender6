Create Barcode

สารเคมีที่ยังไม่มีบาร์โค้ด

คลิกที่การ์ดเพื่อสร้างบาร์โค้ดใหม่

วิธีใช้งาน Create Barcode

คู่มือการใช้งานระบบสร้างบาร์โค้ด

ภาพรวม

หน้า Create Barcode ใช้สำหรับสร้างและจัดการบาร์โค้ดสำหรับอุปกรณ์และสารเคมีในห้องปฏิบัติการ

ฟีเจอร์หลัก

สร้างบาร์โค้ดใหม่

คลิกปุ่ม "New Barcode" เพื่อสร้างบาร์โค้ดใหม่สำหรับอุปกรณ์หรือสารเคมี

ดูบาร์โค้ดทั้งหมด

แสดงบาร์โค้ดทั้งหมดที่สร้างไว้ในรูปแบบการ์ดที่สวยงาม

จัดการบาร์โค้ด

แก้ไข ลบ หรือพิมพ์บาร์โค้ดที่มีอยู่แล้ว

ประวัติการใช้งาน

ติดตามประวัติการสร้างและแก้ไขบาร์โค้ด

วิธีใช้งาน

1. สร้างบาร์โค้ดใหม่

  1. คลิกปุ่ม "New Barcode" ที่มุมขวาล่าง
  2. กรอกข้อมูลที่จำเป็น เช่น ชื่ออุปกรณ์, รหัส, ประเภท
  3. เลือกขนาดและรูปแบบบาร์โค้ด
  4. คลิก "สร้างบาร์โค้ด" เพื่อบันทึก

2. ดูและจัดการบาร์โค้ด

  1. บาร์โค้ดทั้งหมดจะแสดงในรูปแบบการ์ด
  2. คลิกที่การ์ดเพื่อดูรายละเอียด
  3. ใช้ปุ่มแก้ไขหรือลบเพื่อจัดการบาร์โค้ด
  4. คลิกปุ่มพิมพ์เพื่อพิมพ์บาร์โค้ด

3. ค้นหาและกรอง

  1. ใช้ช่องค้นหาเพื่อหาบาร์โค้ดที่ต้องการ
  2. กรองตามประเภทหรือวันที่สร้าง
  3. เรียงลำดับตามชื่อหรือวันที่

เคล็ดลับการใช้งาน

💡 เคล็ดลับ

  • • ใช้ชื่อที่ชัดเจนและเข้าใจง่าย
  • • ตรวจสอบข้อมูลก่อนสร้างบาร์โค้ด
  • • บันทึกบาร์โค้ดที่ใช้บ่อยไว้ในรายการโปรด
  • • ใช้ระบบค้นหาเพื่อหาบาร์โค้ดได้เร็วขึ้น

⚠️ ข้อควรระวัง

  • • ตรวจสอบข้อมูลให้ถูกต้องก่อนบันทึก
  • • ไม่ควรลบบาร์โค้ดที่ใช้งานอยู่
  • • สำรองข้อมูลสำคัญก่อนลบ
  • • ตรวจสอบการพิมพ์ก่อนใช้งานจริง
