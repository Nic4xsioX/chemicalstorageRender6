How to Use the System

  1. Dashboard: View all chemical data and overall statistics in the system.
  2. Chemical Cards: Click a card to view details and manage chemicals.
  3. Add new chemical: Use the "+" floating button to add a new chemical.
  4. Borrow/Return: Use the buttons in the details panel to borrow or return chemicals.
  5. Delete chemical: Use the delete button in the details panel or the floating delete button.
  6. Search: Use the search bar above to find chemicals by name or barcode.
  7. Statistics: See the number of chemicals in each state (expired/expiring soon/in stock).
  8. Notifications: Check for notifications from the Notifications menu.
  9. History: See your work history from the Log Data menu.

Note: The system will automatically log all work by default for checking and tracking.
