How to Use the Log History Page

  1. View log history: The system will show all added/deleted chemical logs.
  2. Search for data: Use the search bar to find logs by chemical name or barcode.
  3. Filter data: Select the type of work (add/delete/all) to view specific data.
  4. Clear filter: Use the "Clear Filter" button to clear filtering and view all data.
  5. Export data: Use the "Export CSV" button to download data as a CSV file.
  6. View details: Click on a log to view more details about the chemical.
  7. Pagination: Use the button below to view data on the next page.

Note: The system will automatically log all work by default for checking and tracking.
