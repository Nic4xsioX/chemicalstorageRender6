How to Use the Notifications Page

  1. View notifications: The system will show all chemicals that are expiring soon or expired.
  2. Search for chemicals: Use the search bar to find chemicals by name or barcode.
  3. Filter notifications: Use the filter buttons (All, Expiring Soon, Expired) to view specific notifications.
  4. Sort notifications: Use the sort dropdown to sort notifications by expiry date, name, or barcode.
  5. View details: Click on a notification to view more details about the chemical.

Note: The system will automatically notify you about chemicals that are expiring soon or expired.
